# Social Housing Manager User Guide

## Overview
The Social Housing Manager role is designed for managers of social housing programs, non-profit housing organizations, and subsidy administrators. This guide helps you manage social housing operations in DingDong BMS.

## Dashboard Overview

The Social Housing Manager Dashboard displays:
- **Total Properties:** Social housing units you manage

## Common Tasks

### Verify Tenant Eligibility for Subsidy
1. Click **Tenants** in the sidebar
2. Select a tenant and view subsidy information
3. Check "Income Verification" date
4. If expired, request new verification documents
5. Approve or deny continuation

### Monitor Grant Spending
1. Go to **Grants** in the sidebar
2. Click on a grant to view details
3. See utilization bar (% of budget spent)
4. View remaining budget and deadline
5. Export spending report if needed

### Generate Social Impact Report
1. Click **Reporting** in the sidebar
2. Select date range
3. Choose report type (Annual, Quarterly, Monthly)
4. View social impact metrics
5. Export as PDF for stakeholders

### Process Subsidy Disbursal
1. Navigate to **Subsidies**
2. Filter by disbursement date
3. Review tenant eligibility status
4. Approve eligible tenants for payment
5. Generate batch payment file
6. Upload to accounting system

## Program Administration

### Adding Tenants to Subsidy Program
1. Verify income eligibility
2. Obtain signed program agreement
3. Set initial subsidy amount
4. Enter start date
5. Configure payment schedule

### Renewing Tenant Eligibility
1. Request updated income documentation
2. Verify continued need for assistance
3. Re-evaluate household composition
4. Adjust subsidy amount if needed
5. Update in system 60 days before expiry

### Program Change Process
1. Document reason for change
2. Obtain appropriate approvals
3. Update subsidy amount
4. Notify tenant in writing
5. Log change with timestamp

## Troubleshooting

### Tenant Not Eligible for Subsidy
**Problem:** Error when trying to add to program  
**Solution:** Verify income documentation on file and household size requirements.

### Grant Deadline Approaching
**Problem:** Notification showing deadline near  
**Solution:** Review spending plan and acceleration strategy with your director.

### Disbursement Failed
**Problem:** Subsidy payment didn't process  
**Solution:** Check tenant bank details, verify account active status, contact finance team.

## FAQ

**Q: How often must I verify income?**  
A: Annual verification required. More frequent if significant income change occurs.

**Q: What happens if tenant income exceeds limit?**  
A: Gradual phase-out period applies. Review your program guidelines for specific intervals.

**Q: Can I manage multiple grant programs?**  
A: Yes. Each grant is tracked separately with its own budget and timeline.

**Q: How is social impact measured?**  
A: Through tenant surveys, housing retention data, and employment verification.

**Q: What compliance documentation is required?**  
A: Varies by program. Check your grant agreement and regulatory requirements annually.

## Key Performance Indicators (KPIs)

Track these metrics monthly:
- **Housing Retention Rate:** % of tenants housed more than 1 year
- **Program Occupancy:** % of available slots filled
- **Grant Utilization:** % of awarded funding spent
- **Tenant Satisfaction:** Average survey score
- **Compliance Rate:** % of audited items compliant
- **Cost per Unit:** Average subsidy cost per tenant per month
