## Common Tasks

### Monitor Portfolio Performance
1. Go to **Dashboard**
2. Review key metrics
3. Click **Portfolio** for detailed view
4. Compare properties side-by-side
5. Identify underperforming assets

### View Revenue Forecasting
1. Navigate to **Analytics**
2. Select date range for forecast
3. View 12-month revenue projection
4. Compare against actual performance
5. Export forecast for board presentation

### Manage Property Staff
1. Click **Staff** in the sidebar
2. View all managers and assignments
3. Click manager to see their properties
4. View performance metrics
5. Schedule training or reviews

### Ensure Compliance
1. Go to **Compliance**

3. Check certification expiration dates

5. Track remediation items

## Troubleshooting

### Property Not Showing in Portfolio
**Problem:** New property acquisition not visible  
**Solution:** Allow 24 hours for system sync. Contact support if not resolved.

### Revenue Forecast Not Accurate
**Problem:** Forecast doesn't match actuals  
**Solution:** Verify historical data is correct. Adjust for known market changes.

### Staff Assignment Error
**Problem:** Can't assign manager to property  
**Solution:** Verify manager is active and not already assigned to 5+ properties.

### Compliance Item Not Clearing
**Problem:** Completed inspection still showing as due  
**Solution:** Upload proof of completion document. System processes within 2 hours.
