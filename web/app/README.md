# DingDong System Modules

This directory contains the core functional modules of the Building Management System.

## Module Structure

- **identity/**: Tenant Onboarding, Visitor Management, Parking.
- **iot/**: Elevators, Gardens, Safety, Emergency protocols.
- **fiscal/**: Billing, Requisition, Commerce.
- **logistics/**: Commute, Room Booking.
- **sustainability/**: Energy, Waste, Air Quality.
- **experience/**: Concierge, Community, Wayfinding.
